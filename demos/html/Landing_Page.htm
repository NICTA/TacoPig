Figure ?? illustrates the structure of the Gaussian process model used by TacoPig.

At the model’s centre is a Gaussian process class instance which contains the key functions for inference and learning as well as the properties such as the training data. Typical examples of this type of class are the tacopig.gp.Regressor and tacopig.gp.ClassifierLaplace which perform regression and classification, respectively.
